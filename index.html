Nice Colour Package

To use the HexaDecimal color package, select a color from the Analogous color theme and click on it. The color code will be automatically copied to your system’s clipboard in hexadecimal format.

Color Theme

It is a useful tool that helps you select and compare colors to find the perfect shade. You can generate new colors by clicking the ‘Generate’ button, and choose a color by clicking on the corresponding flag. To copy the hexadecimal color code, double-click.

Colour Name Package

To use this package, you can hover over the color slide that sets the color in the frame. On clicking the frame, the color name will be copied to the clipboard.

Colour Contrast

The colors white and black are considered to be the most versatile colors as they match well with any other color. In color contrast, the darker version of a color is called a shade, while the lighter version is called a tint.
